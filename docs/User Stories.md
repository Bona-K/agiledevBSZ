## User Stories (Confirmed)

Please feel free to:
- suggest any additional user stories
- point out anything unnecessary
- or propose improvements

---

### 🔹 Account & Access
- As a user, I want to create an account so that I can save my routes and interact with others.
- As a user, I want to log in so that I can access my saved routes.
- As a user, I want to log out so that I can secure my account.
- As a user, I want to change my password so that I can manage my account securely.

---

### 🔹 Explore & Discover
- As a user, I want to explore other users’ routes so that I can get inspiration.
- As a user, I want to search routes by keyword so that I can find relevant plans.
- As a user, I want to filter routes by tags or themes so that I can match my vibe.
- As a user, I want to view top-rated or popular routes so that I can find the best plans.
- As a user, I want to view trending routes so that I can see what others are enjoying.
- As a user, I want to see similar routes so that I can explore alternative plans.
- As a user, I want to see routes near my location so that I can plan conveniently.
- As a user, I want to view route details so that I can understand the full schedule.
- As a user, I want to see locations in order with time information so that I can follow the plan easily.
- As a user, I want to see a helpful message when no results are found so that I know what to do next.

---

### 🔹 Route Interaction (Community)
- As a user, I want to like other users’ routes so that I can support them.
- As a user, I want to leave comments on routes so that I can share my opinions.
- As a user, I want to save/bookmark other users’ routes so that I can revisit them later.
- As a user, I want to share my route via a link so that others can easily access it.
- As a user, I want to duplicate an existing route so that I can customize it for my needs.
- As a user, I want to view a user’s profile so that I can explore more of their routes.
- As a user, I want to follow other users so that I can see their latest routes.

---

### 🔹 Route Creation
- As a user, I want to create a route so that I can plan my day.
- As a user, I want to assign a theme and tags to my route so that others understand the concept.
- As a user, I want to set my route as public or private so that I can control visibility.
- As a user, I want to save a route as a draft so that I can finish it later.
- As a user, I want to receive feedback if required fields are missing so that I can complete my route correctly.
- As a user, I want to generate a route based on my preferences so that I can quickly plan my day.

---

### 🔹 Route Points (Locations)
- As a user, I want to add multiple locations to a route so that I can build a full schedule.
- As a user, I want to reorder locations within a route so that I can optimize my plan easily.
- As a user, I want to upload photos for each location so that I can document my experience.
- As a user, I want to reuse saved locations so that I don’t have to re-enter the same place.
- As a user, I want to edit or remove locations so that I can adjust my route.
- As a user, I want to see estimated travel time between locations so that I can plan realistically.
- As a user, I want to view the best time to visit a location so that I can plan better.

---

### 🔹 Navigation & Map
- As a user, I want to view my route on a map so that I can understand directions visually.
- As a user, I want to open locations in Google Maps so that I can navigate easily.

---

### 🔹 Recommendation & Personalization
- As a user, I want to receive personalized route recommendations so that I can discover routes that match my interests.

---

### 🔹 Route Usage & Experience
- As a user, I want to mark a route as completed so that I can track my experiences.
- As a user, I want to rate a route after completing it so that I can help other users.
- As a user, I want to add personal notes to a route so that I can remember important details.

---

### 🔹 Personal Management
- As a user, I want to view my profile information so that I can see my activity.
- As a user, I want to view my created routes so that I can manage them.
- As a user, I want to edit or delete my routes so that I can update my plans.
- As a user, I want to view routes I saved from others so that I can reuse them.

---

### 🔹 Notifications
- As a user, I want to receive notifications when someone likes my route so that I stay engaged.
- As a user, I want to receive notifications when someone comments on my route so that I can respond.
- As a user, I want to receive notifications about new routes that match my interests so that I don’t miss out.

---

### 🔹 Content Moderation & Quality
- As a user, I want to report inappropriate routes or comments so that the platform remains safe.
- As a user, I want to see highly rated or verified routes so that I can trust the content.

---

### 🔹 System Feedback & Usability
- As a user, I want to see clear error messages so that I understand what went wrong.
- As a user, I want fast loading of routes so that I have a smooth experience.
- As a user, I want the application to be mobile-friendly so that I can use it on any device.
